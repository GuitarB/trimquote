# Contributing to TrimQuote

Thank you for your interest in TrimQuote.

TrimQuote is commercial software developed specifically for automotive upholstery shops.

## Bug Reports

If you've found a bug, please open a GitHub Issue with:

- Steps to reproduce
- Expected behavior
- Actual behavior
- Browser and operating system

Screenshots are always appreciated.

---

## Feature Requests

We welcome ideas that improve the quoting process, shop workflow, customer management, invoicing, reporting, and overall usability.

Before submitting a request:

- Search existing issues first.
- Clearly explain the problem you're trying to solve.
- Include examples whenever possible.

---

## Documentation Improvements

If you notice documentation that is incorrect or unclear, please open an Issue.

---

## Security

Please do not report security vulnerabilities publicly.

Instead, follow the process described in SECURITY.md.

---

## Code Contributions

The TrimQuote application itself is proprietary software.

Public pull requests for the application source code are not accepted at this time.

However, documentation improvements and product feedback are always appreciated.

Thank you for helping improve TrimQuote.
